Polygon Metrics
The polygon metrics form allows you to select which metrics you want to calculate for the change layers created by the overlay tool.

Input: See Robertson et al. 2007, Table 4 for calculations.
a) Number Ratio - number of polygons T2 / number of polygons T1

b) Area Ratio - area of polygons T2 / area of polygons T1

c) Avg Area Ratio - Area Ratio / Number Ratio

d) Event Measure - number of events / number of polygons in T1 and T2

e) Area Event Measure - proportion of area each of the event types make up for each changelayer

f) Four Cone Directional Change - calculate event changes in four cardinal directions using cone method

g) Size Change - calculate local size changes for each polygon group

h) Spatial-temporal history - output a csv file containing change metrics summed for distinct spatial-temporal histories

i) Directed graph - output a directed graph for visualization of spatial-temporal relationships

Output:
a) ChangeLayers - shapefiles created in the workspace directory with attributes for further STAMP analysis
