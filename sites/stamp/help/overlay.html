STAMP Overlay Tool
The STAMP Overlay Tool unions input layers to create change layers that are used for subsequent processing and analysis. Layers should appear in the list in the correct temporal order. If they are not, cancel the form and rearrange in ArcMap’s Table of Contents.

Input:
a) Input Layers - input polygons layers

b) Workspace Directory - directory to store all of the output files creating by all STAMP tools. Previous STAMP output and any shapefiles in this directory will be deleted.

c) Distance Threshold - the maximum distance in map units for relating polygons from two time periods

d) T1 / T2 Classification - indicates whether movement classifications should be looking forward (T2 relative to T1) or backward (T1 relative to T2). If unsure, select T2.

Output:
a) ChangeLayers - shapefiles created in the workspace directory with attributes for further STAMP analysis
