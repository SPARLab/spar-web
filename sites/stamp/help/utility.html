Polygon Utility
The polygon utility checks the input polygons to see if they are spatially exclusive. Since all polygons are checked individually, this can be a time consuming process if there are many polygons per layer.

Input:
a) Input Layers - the layers you want to process with   STAMP. They should be polygon layers added to the map.

Output:
a) Message Box - Telling you if input layers are spatially exclusive.

b) Selection Set - If there are violating features, they will be selected in the map.
