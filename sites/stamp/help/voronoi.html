Voronoi Polygons for Polygonal Data
Voronoi polygons for input polygons are used for computing directional relationships between polygons. Voronoi polygons need to be computed within a bounded area, so this form allows you to select a layer as a study area or to use the bounding box of all input layers as the study area. The bounding box approach is much faster because it reduces the number of input vertices.

Input:
a) Study Area Layer / Bounding Box - selection of method to use for study area of voronoi polygons
Output:
a) Voronoi Shapefiles - Voronoi shapefiles are created in the workspace directory (intermediate voronoi files are added to the map during processing)
